Database

  1. Temporary Storage

    1. Local Storage
      • Works in browser
      • deleted when user deletes and can't define limit time
      • shares on all windows(tab)
    2. Cookie
      • Works in both browser and server
      • deleted when user deletes or reaches limit time
      • shares on all windows(tab)
    3. Session
      • Works in browser(js)
      • Delete when tab is closed
      • Not share windows - Only works in its own tab
